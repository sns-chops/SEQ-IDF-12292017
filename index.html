SEQUOIA detector calibration results 12/29/2017

Detector geometry calibration was performed using powder Si and C60 calibration datasets to better determine the positions and orientation of detector packs in the SEQUOIA instrument.

This work was performed using the notebooks and code in sns-chops/detcalib/.../SEQ.

Progress reports for this work can be found in sns-chops/SEQ-IDF-12292017/.../reports.
